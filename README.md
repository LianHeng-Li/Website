# Academic Group Website Template

This repository is structured for GitHub Pages with Jekyll.

## Main folders

- `index.md`: root redirect for GitHub Pages
- `home/index.md`: homepage
- `news/index.md`: news page
- `publications/index.md`: publications page
- `projects/index.md`: projects page
- `contact/index.md`: contact page
- `_layouts/default.html`: shared page layout
- `assets/css/style.css`: site style
- `images/`: images and figures

## How to edit

1. Replace the text in each `index.md` file with your own content.
2. Update `_config.yml` for the site title, tagline, and footer text.
3. Replace `images/profile.jpg` with your own photo or lab image.
4. Push the repository to GitHub Pages.

## Notes

- The pages use a white academic style with black, blue, and light purple accents.
- Each major section is stored in its own folder for easier future editing.
